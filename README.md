# Spear-of-Athena
Spear of Athena — мифологический слот, основанный на силе богини Афины. Игра сочетает высокую волатильность, мощные множители и бонусы с копьём, способным умножать выигрыш в десятки раз. Эпический дизайн, насыщенные эффекты и сильная механика делают Spear of Athena идеальным выбором для любителей мифологических слотов с большим потенциалом.
